# ICTAK LMS

A full-stack Learning Management System (LMS) developed for managing courses, trainers, students, and administrators.

## Overview

ICTAK LMS is a web-based platform that streamlines learning management by providing separate functionalities for administrators, trainers, and students.

The application includes authentication, course management, student management, and role-based access control.

---

## Features

- User Authentication
- Role-based Authorization
- Course Management
- Student Management
- Trainer Management
- Admin Dashboard
- Responsive UI
